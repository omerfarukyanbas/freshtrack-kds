FreshTrack paneli buradan çalışmaz

Adres çubuğunda file:/// görüyorsan sayfa bilgisayarındaki ham index.html dosyasıdır. Vite, /src/main.tsx dosyasını bu yolla sunamaz; bu yüzden ekran boş kalır.

Doğru kullanım:

  1. Node.js kurulu olsun (npm ile birlikte).
  2. Terminalde klasöre gir: frontend
  3. Çalıştır: npm install sonra npm run dev
  4. Tarayıcıda aç: http://localhost:5173

Üretim için: npm run build ardından npm run preview ile yine bir sunucu adresinden aç.
